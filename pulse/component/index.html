W3C Validation Services

While it is still best practice for developers to continue to validate their markup, please note that as of WCAG 2.2, 4.1.1 Parsing (Level A) has been removed. As a result, this is no longer a required QA test. Issues previously flagged under this guideline are generally caught through other testing methods.
